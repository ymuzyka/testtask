## Start project

- create ```.env``` file and set db config
 
Default config:

> DB_CONNECTION=mysql
>   
> DB_HOST=mysql
>
> DB_PORT=3306
> 
> DB_DATABASE=testtask
> 
> DB_USERNAME=root
> 
> DB_PASSWORD=root

- install packages ``` composer install ```
- start project in docker ``` docker-compose up -d ```
